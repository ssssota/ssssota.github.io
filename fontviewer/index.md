# Fontviewer

![Screenshot](https://raw.githubusercontent.com/ssssota/fontviewer/master/Screenshot.png)
所有フォントの一覧を表示します．

## Function

機能は次のとおりです．

- 所有フォントの一覧表示
- フォント検索
- 表示テキストの変更
- 背景色の変更
- 文字色の変更
- 文字サイズの変更
